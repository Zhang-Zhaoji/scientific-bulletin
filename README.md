# Neuroscience Bulletin

Neuroscience Bulletin is a Wechat public account that looks up recent neuroscience articles and shares them with you. We use a crawler to search their titles and abstracts from the best neuroscience-related journals, and summarize them through language models. Currently, we are using the Qwen3-4B-Instruce-2507 model, which is one of the SOTA small language models.

This repository contains the code and some of the data of the Neuroscience Bulletin project. We wish to make the project open-source and accessible to everyone. If you have any questions or suggestions, please feel free to contact us.
